Template Literals ← Big Upgrade

// Old way
"Hello, " + name + "! Age: " + age

// Template literal — backtick + ${}
`Hello, ${name}! Age: ${age}`

// Any expression inside ${}
`Price: $${price * 1.08}`
`Is adult: ${age >= 18}`

// Multiline — no \n needed
`Name: ${name}
Email: ${email}`

The 6 Falsy Values

false 0 "" null undefined NaN

Everything else is truthy — including "0", "false", [], {}

typeof

typeof "hello"    → "string"
typeof 42         → "number"
typeof true       → "boolean"
typeof undefined  → "undefined"
typeof null       → "object" ⚠️

⚠️ typeof null === "object" is a historic JS bug. Use === null to check for null.
